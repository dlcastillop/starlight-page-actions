Starlight Page Actions comes with a list of Agent Skills that define best practices for working with this plugin.

These skills are useful for AI agents like Claude Code, Codex, or Cursor.

## Installation

```sh
npx skills add https://github.com/dlcastillop/starlight-page-actions --skill starlight-page-actions
```
